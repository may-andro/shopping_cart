# Shopping Cart
A demo application for android arch component with MVVM using Hilt Android. The application uses pagination and motion layout. 

Features Included: 
1. MVVM Clean Arch
2. SOLID Principles
3. HILT Dependency
4. Motion Layout
5. Jetpack Components
6. Coroutines and Flow
7. Paging 3
8. Retrofit and Room
9. Orientation Haneling
